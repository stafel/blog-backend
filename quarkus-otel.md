## Dependency

Damit mit opentelemetry gearbetet werden kann muss die dependency ```quarkus-opentelemetry``` im pom.xml aufgenommen werden.

```
<dependency>
    <groupId>io.quarkus</groupId>
    <artifactId>quarkus-opentelemetry</artifactId>
</dependency>
```
